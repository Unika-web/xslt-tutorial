С XSL вы можете свободно модифицировать исходный текст. Так с помощью преобразований 1 и 2 можно получить различный результат из одного исходного файла.

Преобразование 1

Исходный XML
<source>

<title>XSL</title>
<author>John Smith</author>

</source>

Результат
<h1>XSL</h1>
<h2>John Smith</h2>

Представление HTML

XSL

John Smith
Преобразование XSLT
<xsl:stylesheet version = '1.0'
     xmlns:xsl='http://www.w3.org/1999/XSL/Transform'>

<xsl:template match="/">
     <h1>
          <xsl:value-of select="//title"/>
     </h1>
     <h2>
          <xsl:value-of select="//author"/>
     </h2>
</xsl:template>


</xsl:stylesheet>


Преобразование 2

Исходный XML
<source>

<title>XSL</title>
<author>John Smith</author>

</source>

Результат
<h2>John Smith</h2>
<h1>XSL</h1>

Представление HTML

John Smith

XSL
Преобразование XSLT
<xsl:stylesheet version = '1.0'
     xmlns:xsl='http://www.w3.org/1999/XSL/Transform'>

<xsl:template match="/">
     <h2>
          <xsl:value-of select="//author"/>
     </h2>
     <h1>
          <xsl:value-of select="//title"/>
     </h1>
</xsl:template>


</xsl:stylesheet>
